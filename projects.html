I have worked on many different projects during my entire academic career. The most recent one is Food Truck Locator, which I have worked on during last semester. This application can be used by food trucks to indicate their everyday location and by customers to track which food trucks are nearby and get their daily updated location. The frontend was developed using React and external modules, while backend is implemented using Rest APIs with Spring and MySQL to store the data.
